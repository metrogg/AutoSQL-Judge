#### Pandas 在本项目中的角色

- **Pandas 是什么？**
  - Pandas 是 Python 里做**表格/数据分析**的核心库，可以把数据读成 `DataFrame`（有行有列的表），支持过滤、排序、分组统计等操作。
- **在本项目中具体做什么？**
  - 将“标准答案 SQL”和“用户提交 SQL”的执行结果分别读成两个 DataFrame；
  - 对结果进行统一排序、重置索引；
  - 使用 DataFrame 的 `equals` 能力做**结果集级别的严格对比**，从而实现“语义级判题”，而不是简单的字符串匹配。
- **为什么适合用在课设/原型阶段？**
  - 写法直观，几行代码就能完成结果集比对，开发效率高；
  - 非常适合小到中等规模的数据集，便于调试和讲解判题逻辑。
- **局限性与演进空间（可以写在答辩/简历里）**
  - 局限：当结果集非常大、并发很高时，Pandas 的内存开销会变大，更适合用数据库自身的差集/EXCEPT 运算来判题；
  - 演进：后续可以将 `judge_sql` 抽象成独立判题服务，内部从“Pandas 比对”替换为“数据库差集比对”或“专门的评测引擎”，以支撑更大规模的在线评测平台。
